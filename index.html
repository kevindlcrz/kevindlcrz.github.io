Reflection and Learnings

This semester, I dove into the world of managing enterprise servers, where we focused on handling servers from afar using Linux and CentOS. Getting comfortable with command line operations was like learning a new language. We configured, monitored, and fixed servers, gaining a deep understanding of how they work. Ansible, a tool for automating tasks, made things more efficient, especially for repetitive jobs. Exploring Docker taught me about containerization, a cool way to manage and deploy applications. Beyond the technical stuff, this journey taught me to be resilient and adaptable when dealing with complex server setups. Now, as the semester ends, I not only understand server management but also discovered my ability to tackle challenges in the real world.
